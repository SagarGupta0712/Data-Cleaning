# Sales Data Cleaning & Analysis Project

## 📌 Project Overview
This project demonstrates an end-to-end **data cleaning and exploratory data analysis (EDA)** workflow using Python.  
The dataset represents sales transactions containing missing values, duplicates, and inconsistent data — simulating a real-world business scenario.

The goal is to clean the data, perform analysis, and extract actionable insights for business decision-making.

---

## 🛠 Tools & Technologies
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn
